QA Automation Testing Project
Overview
This project demonstrates automated testing of a web application's login functionality using Selenium WebDriver with Java. It reflects a practical understanding of software testing concepts, including functional validation, test execution, and result verification.

Objective
To validate the login functionality of a web application by automating user actions and verifying the expected outcome against actual results.

🛠️ Tech Stack
Programming Language: Java
Automation Tool: Selenium WebDriver
Build Tool: Maven
Driver Management: WebDriverManager
Test Scenario: Login Functionality
Test Case: Valid Login

Steps:

Navigate to the login page
Enter valid credentials
Click on the login button
Verify successful login message
Expected Result: User should be successfully logged in and redirected to a secure area.

Execution
The test script performs the following actions:

Launches the browser
Navigates to the application URL
Inputs valid login credentials
Submits the login form
Validates the success message
Outputs the result in the console
Output
On successful execution, the console displays:
TEST PASSED
