🧪 QA Automation Testing Project – Login Functionality
📌 Overview

This project demonstrates automated testing of a web application's login functionality using Selenium WebDriver with Java. It showcases practical knowledge of software testing concepts including test execution, validation, and result verification.

🎯 Objective

To validate the login functionality of a web application by:

Automating user actions

Verifying expected outcomes against actual results

🛠️ Tech Stack

Programming Language: Java

Automation Tool: Selenium WebDriver

Build Tool: Maven

Driver Management: WebDriverManager

🧪 Test Scenario

Feature: Login Functionality
Test Case: Valid Login

🔄 Test Steps

Navigate to the login page

Enter valid credentials

Click on the login button

Verify successful login message

✅ Expected Result

User should be successfully logged in

User should be redirected to a secure/dashboard page

⚙️ Execution Flow

The test script performs the following actions:

Launches the browser

Navigates to the application URL

Inputs valid login credentials

Submits the login form

Validates the success message

Prints result in console

📊 Output

On successful execution: TEST PASSED is displayed in the console

On failure: Appropriate error message is shown
